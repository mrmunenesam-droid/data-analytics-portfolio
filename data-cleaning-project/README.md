

# Data Cleaning & Preparation Project

## Problem
The dataset contained missing values, duplicates, inconsistent formats, and text inconsistencies that made analysis unreliable.

## Dataset
Sales transaction data with order details, customer information, product data, and regions.

## Tools Used
- Microsoft Excel
- Power Query
- Data Validation

## Cleaning Steps
- Removed duplicate records
- Standardized date formats
- Trimmed and standardized text fields
- Handled missing values
- Corrected data types

## Outcome
Produced a clean, analysis-ready dataset with consistent formatting and improved data quality.
